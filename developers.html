Quick Start

Get your API key and make your first request in minutes:

cURL 
curl -X GET "https://api.bitorzo.com/v1/market-data/realtime-prices?symbols=BTC,ETH" \
  -H "Authorization: Bearer YOUR_API_KEY"

Getting Started

Prerequisites

  • Basic understanding of RESTful APIs
  • Knowledge of HTTP requests and responses
  • Familiarity with JSON data format
  • Development environment with HTTP client (cURL, Postman, or programming language)

1. Create an Account

Sign up for a Bitorzo developer account to get your API credentials:

Create Developer Account

2. Get Your API Key

After registration, navigate to your dashboard to generate API keys:

  1. Log in to your developer dashboard
  2. Navigate to "API Keys" section
  3. Click "Generate New Key"
  4. Store your key securely - it won't be shown again

3. Make Your First Request

Test your API key with a simple request:

cURL 
curl -X GET "https://api.bitorzo.com/v1/market-data/realtime-prices?symbols=BTC" \
  -H "Authorization: Bearer YOUR_API_KEY"
Node.js 
const axios = require('axios');

const response = await axios.get('https://api.bitorzo.com/v1/market-data/realtime-prices', {
  params: { symbols: 'BTC' },
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});

console.log(response.data);
Python 
import requests

response = requests.get(
    'https://api.bitorzo.com/v1/market-data/realtime-prices',
    params={'symbols': 'BTC'},
    headers={'Authorization': 'Bearer YOUR_API_KEY'}
)

print(response.json())

Authentication

All API requests must be authenticated using your API key. Include your key in the Authorization header:

HTTP Header 
Authorization: Bearer YOUR_API_KEY

Security Best Practices

  • Never expose your API key in client-side code
  • Use environment variables to store API keys
  • Rotate your keys regularly
  • Use IP whitelisting for production environments

Rate Limiting

API requests are rate limited to ensure fair usage:

Create Payment Request

Stable
POST /api/create-payment-request

Creates a new cryptocurrency payment request with a unique payment ID and receiving address.

Request Body

Parameter Type Required Description
amount number Yes Payment amount in specified currency
currency string Yes Currency code (ETH, BTC, USDT, etc.)
callbackUrl string No Webhook URL for payment notifications
metadata object No Custom data to attach to the payment

Example Request

cURL 
curl -X POST "https://api.bitorzo.com/api/create-payment-request" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 0.5,
    "currency": "ETH",
    "callbackUrl": "https://yourapp.com/webhooks/payment",
    "metadata": {
      "orderId": "12345",
      "customerId": "user_789"
    }
  }'
Node.js 
const axios = require('axios');

const payment = await axios.post('https://api.bitorzo.com/api/create-payment-request', {
  amount: 0.5,
  currency: 'ETH',
  callbackUrl: 'https://yourapp.com/webhooks/payment',
  metadata: {
    orderId: '12345',
    customerId: 'user_789'
  }
}, {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

console.log('Payment Request:', payment.data);

Webhook Security

Verify webhook signatures to ensure requests are coming from Bitorzo.

Signature Verification

All webhook requests include a signature in the X-Bitorzo-Signature header.

Node.js 
const crypto = require('crypto');

function verifyWebhookSignature(payload, signature, secret) {
  const hash = crypto
    .createHmac('sha256', secret)
    .update(JSON.stringify(payload))
    .digest('hex');
  
  return hash === signature;
}

// In your webhook handler
app.post('/webhook', (req, res) => {
  const signature = req.headers['x-bitorzo-signature'];
  const isValid = verifyWebhookSignature(req.body, signature, WEBHOOK_SECRET);
  
  if (!isValid) {
    return res.status(401).send('Invalid signature');
  }
  
  // Process the webhook
  console.log('Webhook event:', req.body.event);
  res.sendStatus(200);
});
Python 
import hmac
import hashlib
import json

def verify_webhook_signature(payload, signature, secret):
    expected = hmac.new(
        secret.encode(),
        json.dumps(payload).encode(),
        hashlib.sha256
    ).hexdigest()
    
    return hmac.compare_digest(expected, signature)

# In your webhook handler
@app.route('/webhook', methods=['POST'])
def webhook():
    signature = request.headers.get('X-Bitorzo-Signature')
    payload = request.json
    
    if not verify_webhook_signature(payload, signature, WEBHOOK_SECRET):
        return 'Invalid signature', 401
    
    # Process the webhook
    print(f"Webhook event: {payload['event']}")
    return '', 200
